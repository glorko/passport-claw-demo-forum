# passport-claw-demo-forum

Minimal Node HTTP server: `GET/POST /api/posts`, `/api/info`, `/healthz`. Agent detection per `architecture/demo-board.md`.

See `../passport-claw-dev/.env.example` — `VERIFIER_BASE_URL`, `VERIFIER_API_KEY`, `PASSPORT_DOCS_URL`, `PORT`.
